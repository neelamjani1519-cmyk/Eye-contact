Eye Contact — `ww` (vw) sizes demo

These tokens use viewport width units (`vw`) as the "ww" sizing system.

Modifier classes

xs (2.5vw)
sm (3.5vw)
md (5vw)
lg (8vw)
xl (12vw)

Attribute API

data-size="md"
data-size="lg"

Usage

<!-- class API -->
<div class="eye-contact eye-contact--lg"><!-- ... --></div>

<!-- attribute API -->
<div class="eye-contact" data-size="sm"></div>
